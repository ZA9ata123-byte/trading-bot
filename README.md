# 🤖 Trading Bot — RSI Divergence + EMA

بوت تداول آلي على Binance

## التثبيت
pip install -r requirements.txt

## الإعداد
1. انسخ .env وحط مفاتيح Binance
2. غير PAPER_TRADING=True للتجربة

## التشغيل
python3 bot.py

## الميزات
- RSI Divergence + EMA
- Trailing Stop ذكي
- فلتر الأخبار
- قاعدة 24 ساعة
- أفضل 300 عملة تلقائياً
